Example script showing how to use the ProgressBar component. Attach this to a GameObject with a ProgressBar component to see it in action.

AUTO TEST MODES:

  • Positive Value Modifier: Fills progress bar from current to maximum, then resets to minimum and repeats
  • Negative Value Modifier: Drains progress bar from current to minimum, then sets to maximum and repeats

The direction (fill/drain) is automatically determined by the sign of the Value Modifier. When Loop Test is enabled with negative modifier, it creates a "fill → drain → fill → drain" cycle.
